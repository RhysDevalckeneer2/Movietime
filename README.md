## Installation

Make sure you have set up a React Native environement like [here](https://reactnative.dev/docs/environment-setup)

```
  $ git clone
  $ cd movietime
```
Install dependencies
```
  $ npm install
```
If you're running iOS, make sure you install the pods
```
  $ cd ios
  $ pod install
```
Run Android or iOS
```
  $ npm run android
```
```
  $ npm run ios
```
